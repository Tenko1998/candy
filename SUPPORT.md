# Support AiCos9

AiCos9 is an open-source project built slowly with care.

If this project helps you, inspires you, or gives you hope for a more personal and continuous AI companion experience, you can support its development through optional donations.

Support may help with:

- development tools
- API usage for testing
- documentation
- design assets
- prototype work
- local model experiments
- future community and plugin research

AiCos9 will remain open source. Donations are always optional, and they do not change the project's core philosophy: users should be able to bring their own model, keep control over their own companion home, and build with clear boundaries.
