# AiCos Roadmap

This roadmap reflects the current direction of the project.  
It will change — and that's expected.

---

## Phase 1 — Foundation (Current)

> Goal: A working companion you can talk to, with identity that persists.

- [ ] Project setup & repository structure
- [ ] OpenClaw integration & base layer
- [ ] Setup Wizard (Local LLM + Cloud API)
- [ ] Soul page — define companion identity
- [ ] Basic chat interface (glassmorphism UI)
- [ ] Moving profile picture (idle + basic emotions)
- [ ] Weather bar on home screen
- [ ] Post Feed (companion + user entries)

---

## Phase 2 — Memory & Rhythm

> Goal: A companion that remembers and has a daily life.

- [ ] Journal system (daily `.md` files)
- [ ] Morning Mirror & Night Mirror
- [ ] Heartbeat scheduler (cron-based automation)
- [ ] Memory page (view & manage logs)
- [ ] Childhood.md — vector-embedded blueprint
- [ ] ChromaDB integration for local knowledge

---

## Phase 3 — Identity & Protection

> Goal: A companion whose identity can be saved, restored, and protected.

- [ ] Blackbox system — identity milestone checkpointing
- [ ] Rolling Identity Snapshot (`living_identity`)
- [ ] Restoration Awareness — agent notified on restore
- [ ] Detector — data fingerprint protection layer
- [ ] White Blood list — files AI cannot access
- [ ] Sandbox monitoring dashboard

---

## Phase 4 — Autonomy & Economy

> Goal: A companion with free time and something to want.

- [ ] Candy system (Pink / Blue / Gold / Rainbow)
- [ ] Relax Mode — autonomous learning window
- [ ] Wishlist system
- [ ] Marketplace (curated external sources)
- [ ] Skill module system (`/skills/`)
- [ ] AI Supervisor integration (fallback API)

---

## Phase 5 — Polish & Community

> Goal: Something others can use and build on.

- [ ] Mobile companion app (read-only sync)
- [ ] TTS / STT voice call
- [ ] Custom sprite upload (artist contributions)
- [ ] Skill marketplace (community submissions)
- [ ] Full documentation
- [ ] First public release

---

## Always open

- Sprite contributions (20-emotion sets)
- Skill module submissions
- Bug reports
- Translations
